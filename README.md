## 介绍

tesla使用koa2+react+webpack4+gulp,以及npm集成管理工具,登录 [DIY(http://diy.liuxue.com)](http://diy.liuxue.com) ,体验功能.

## 目录结构
```
tesla/
├── config  //  webpack插件，开发，生产配置文件
├── dist   //  编译生成打包文件
├── docs   //   jsdocs文档
├── logs    //  日志
├── src/
│   ├── server/
│   ├── webapp/
├── tests/   //  测试
```

### 本地运行说明

```
node版本>v7
npm install
npm run webpack
npm run gulp
npm run start
```
